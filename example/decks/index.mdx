# Slide decks with style

Using an emoji favicon with Gatsby themes and MDX Deck

---

1. Add `gatsby-theme-mdx-deck` to your Gatsby config
2. Configure `gatsby-plugin-emoji-favicon` in your Gatsby config
3. Create some slides in `decks/index.mdx`

---

```js
// gatsby-config.js
module.exports = {
  plugins: [
    'gatsby-theme-mdx-deck',
    {
      resolve: 'gatsby-plugin-emoji-favicon',
      options: {
        emoji: 'ðŸ›¶'
      }
    }
  ]
};
```
